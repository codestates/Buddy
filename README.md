## About
혼자 부담 없이 얘기하고 싶은 친구를 찾고 계시나요?

심심한 날 익명의 친구들과 대화를 나누고 싶으신가요?

버디(Buddy)에서는 익명의 친구들과 대화할 수 있는 채팅방이 준비되어 있습니다.

일상의 무료함을 달래고 싶을 땐 저 버디(Buddy)를 찾아주세요!
